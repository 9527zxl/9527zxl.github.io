环境名称无所谓，符合要求即可

购买完成之后即可进入控制台

然后来到安全配置中，将自己的网站域名添加到 web安全域名

命名为 package.json，按下回车，并且复制以下代码 { "dependencies": { "twikoo-func": "1.3.0" } }

开启 twikoo 评论管理面板
